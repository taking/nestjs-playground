## Description
- 추후 NestJS를 활용하여 프로젝트를 진행 시, 사용할 NestJS PlayGround이다.
- yarn 대신 [bun](https://bun.sh/)을 사용한다.

## Installation

```bash
$ bun install
```

## Running the app

```bash
# development
$ bun run start

# watch mode
$ bun run start:dev

# production mode
$ bun run start:prod
```

## Test

```bash
# unit tests
$ bun run test

# e2e tests
$ bun run test:e2e

# test coverage
$ bun run test:cov
```
